# audio-extract
Extract audio (to mp3, m4a, or wav) from an MP4 video file. Support for more input file types coming soon!

You can install the dependencies with ```pip install -r requirements.txt```. After installing, run ```python -c "import imageio_ffmpeg; print(imageio_ffmpeg.get_ffmpeg_exe())"``` to make sure that the Python-specific ffmpeg binary is installed.
